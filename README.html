如何使用

浏览程序

打开页面后，左侧会显示程序分类目录，点击分类即可展开查看该分类下的所有程序。右侧显示程序卡片，每张卡片包含程序名称、图标和文件名。

搜索程序

在页面顶部的搜索框中输入关键词，即可快速过滤出需要的程序或分类。

下载程序

点击程序卡片，页面会弹出下载进度窗口，显示实时下载进度。下载完成后，浏览器会自动弹出保存对话框。

更新与删除程序

程序更新与删除

访问\\QITV1534\共享文件\安装目录\X_FastDownload在此文件夹下写入或删除文件等待35S等待服务器程序自动同步并更新后端数据文件，刷新后即可看到最新的程序列表

服务器不支持文件夹下载，若工具是以文件夹形式保存的，需要手动压缩为Zip包后再上传

目录结构

App快速下载/
├── X_FastDownload                # 存放程序的文件夹（核心） 共享路径\\QITV1534\APPFastDownload\Main
│   ├── CPU/             # 一级分类示例
│   │   ├── 子分类A/     # 二级分类（可选）
│   │   │   └── 深层/    # 三级分类（最深层级）
│   │   │       └── tool.exe
│   │   └── cpuz.exe
│   ├── GPU/
│   ├── 内存/
│   └── 磁盘工具/

注意： 子目录最多嵌套 3 层（Main 下第 1 层 → 第 2 层 → 第 3 层），超出层级的目录将被忽略。

支持的文件格式

工具箱支持以下格式的程序文件：

格式说明
.exe可执行程序
.msiWindows 安装包
.zip压缩包
.rar压缩包
.7z压缩包
.iso光盘镜像
.bat批处理脚本
.cmd命令脚本
.ps1PowerShell 脚本
.reg注册表文件
.txt文本文件

功能说明

功能说明
分类导航左侧树形目录，支持展开/收起，方便按类别浏览程序
搜索过滤顶部搜索框，输入关键词即可实时过滤分类与程序
下载进度点击下载后弹出进度窗口，实时显示下载速度和剩余大小，支持中途取消
深色/亮色模式自动跟随系统主题，也可点击右上角按钮手动切换
程序图标自动显示程序的图标，方便快速识别
