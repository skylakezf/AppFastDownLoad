

基于 H5 + CSS 的离线工具箱，提供图吧工具箱风格的程序管理与快速下载体验。

---

## 目录结构

```
App快速下载/
├── Main/                # 存放程序的文件夹（核心）
│   ├── CPU/             # 一级分类示例
│   │   ├── 子分类A/     # 二级分类（可选）
│   │   │   └── 深层/    # 三级分类（最深层级）
│   │   │       └── tool.exe
│   │   └── cpuz.exe
│   ├── GPU/
│   ├── 内存/
│   └── 磁盘工具/
├── generate.py          # Python 扫描脚本
├── 更新程序.bat          # 一键运行扫描脚本
├── data.js              # 自动生成的前端数据文件（勿手动编辑）
├── index.html           # 前端页面（直接用浏览器打开）
└── README.md            # 本说明文件
```

## 快速开始

### 1. 放置程序

将需要展示的程序（`.exe` / `.msi` / `.bat` / `.cmd` / `.zip` / `.rar` / `.7z`）放入 `Main` 文件夹下的对应分类目录中。

**示例：**

```
Main/
├── CPU/
│   └── cpuz.exe
├── GPU/
│   └── GPU-Z.exe
├── 内存/
│   └── memtest.exe
└── 磁盘工具/
    └── DiskGenius.zip
```

### 2. 生成数据

运行 `更新程序.bat` 或手动执行：

```powershell
python generate.py
```

> **依赖：** 如需提取 `.exe` 图标，请安装 `pywin32` 和 `Pillow`：
> ```
> pip install pywin32 Pillow
> ```
> 未安装依赖时脚本仍可正常运行，但程序图标将使用默认终端图标替代。

### 3. 打开页面

直接用浏览器打开 `index.html` 即可使用。

## 程序命名建议

为了方便识别与展示，建议遵循以下命名规范：

### 基本格式

```
程序名 描述.扩展名
```

**示例：**
- `cpuz 处理器检测.exe`
- `GPU-Z 显卡信息采集.exe`
- `DiskGenius 磁盘分区.zip`
- `HWiNFO64 硬件检测.exe`

### 命名规则

| 规则 | 说明 |
|------|------|
| **用空格分隔** | 程序名与描述之间用空格分隔，前端会将整个文件名（含描述）作为程序名称展示 |
| **保留版本号** | 如 `GPU-Z.2.56.0` 或 `v2.1.5`，方便区分不同版本 |
| **添加简短描述** | 用 2-5 个中文字描述程序用途，如 `检测`、`清理`、`分区`、`` |
| **避免特殊符号** | 文件名中避免使用 `/ \ : * ? " < > |` 等非法字符 |
| **使用中文分类名** | 文件夹名直接用中文命名，如 `CPU`、`内存`、`磁盘工具` |

### 分类建议

以下是推荐的分类示例：

| 文件夹名 | 放入的程序类型 |
|----------|--------------|
| `CPU` | CPU 检测、性能测试、稳定性测试 |
| `GPU` | 显卡检测、GPU 测试、驱动管理 |
| `内存` | 内存检测、内存清理 |
| `磁盘工具` | 分区工具、磁盘检测、数据恢复 |
| `系统工具` | 系统信息、任务管理、注册表 |
| `网络工具` | 网络检测、测速、抓包 |
| `显示` | 显示器测试、屏幕检测 |
| `外设` | 键盘、鼠标、USB 检测 |

### 子分类（可选）

如果某分类下程序较多，可再建子目录分层：

```
Main/
└── 磁盘工具/
    ├── DiskGenius/
    │   ├── DiskGenius.exe
    │   └── DGFileViewer.exe
    ├── 分区/
    │   └── partition.exe
    └── 检测/
        └── diskcheck.exe
```

> **注意：** 子目录最多嵌套 3 层（Main 下第 1 层 → 第 2 层 → 第 3 层），超出层级的目录将被忽略。

## 常见问题

**Q: 放入新程序后页面没有更新？**
A: 需要重新运行 `更新程序.bat` 生成新的 `data.js`，然后刷新浏览器页面。

**Q: 图标显示为默认终端图标？**
A: 请确保已安装 `pywin32` 和 `Pillow`，重新运行脚本即可提取图标。

**Q: 支持哪些文件格式？**
A: `.exe`、`.msi`、`.bat`、`.cmd`、`.zip`、`.rar`、`.7z`
